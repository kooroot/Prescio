## 3. 주요 결정사항

### 리더보드
- 온체인 직접 조회 + Multicall 최적화 (초기), 추후 인덱서 연동
- 티어별 색상: Bronze(#B45309), Silver(#9CA3AF), Gold(#EAB308), Diamond(#22D3EE)
- 30초 폴링 주기로 실시간 업데이트

### addStake V5
- 함수 시그니처: `addStake(uint256 amount, bool extendLock)`
- `extendLock=false` 기본값, 만료 시 최소 7일 락업 재설정
- `firstEligibleEpoch`, `startTime` 유지 (보상 자격 보존)
- CEI 패턴 + nonReentrant로 보안 강화
